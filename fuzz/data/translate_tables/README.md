Vendored liblouis tables used by the `translate*` fuzz targets.

These files are a pinned copy of the selected translate-table set and its
transitive `include` closure. The fuzz harness reads from this directory so it
does not depend on a separate `liblouis/tables` checkout at runtime.

`fuzz_maximal.ctb` is a local synthetic table layered on top of that vendored
set. It is not generated by the sync script and is preserved across refreshes.

To refresh them from a local `liblouis/tables` clone:

```sh
./fuzz/vendor-translate-tables.sh
```
